## 1. 系统整体工作流程

```mermaid
flowchart TD
    Start([开始]) --> Login[用户登录/注册]
    Login --> Auth{认证通过?}
    Auth -->|否| Login
    Auth -->|是| Dashboard[进入仪表盘]
    
    Dashboard --> CreateProj[创建项目]
    CreateProj --> ProjDetail[项目详情页]
    
    ProjDetail --> ReqAnalysis[需求分析]
    ReqAnalysis --> ReqConfirm[需求确认]
    ReqConfirm --> ArchDesign[架构设计]
    ArchDesign --> ArchReview[架构评审]
    ArchReview --> TraceMap[追溯映射]
    TraceMap --> DocGen[文档生成]
    DocGen --> End([结束])

    style End fill:#e1f5fe

    style ArchDesign fill:#fff3e0
```

## 2. 用户认证流程

```mermaid
flowchart TD
    Start([开始]) --> Login[输入用户名/密码]
    Login --> Validate[验证输入]
    Validate --> CheckUser{用户存在?}
    CheckUser -->|否| Error1[返回错误: 用户不存在]
    CheckUser -->|是| CheckPwd{密码正确?}
    CheckPwd -->|否| Error2[返回错误: 密码错误]
    CheckPwd -->|是| CheckActive{账户激活?}
    CheckActive -->|否| Error3[返回错误: 账户已禁用]
    CheckActive -->|是| GenToken[生成 JWT Token]
    GenToken --> ReturnToken[返回 access_token + refresh_token]
    ReturnToken --> End([结束])
    
    Error1 --> End
    Error2 --> End
    Error3 --> End
    
    style Start fill:#e1f5fe
    style End fill:#e1f5fe
    style GenToken fill:#e8f5e9
    style ReturnToken fill:#e8f5e9
```

## 3. AI 需求分析工作流程（核心）

```mermaid
flowchart TD
    Start([开始]) --> Upload[上传需求文档/输入需求]
    Upload --> ParseDoc{文档解析}
    ParseDoc -->|有文档| ExtractText[提取文本内容]
    ParseDoc -->|手动输入| ManualInput[手动输入需求]
    
    ExtractText --> AIExtract
    ManualInput --> AIExtract
    
    AIExtract[AI 需求提取<br/>LLM Call #1] --> ParseResult[解析 JSON 结果]
    ParseResult --> SaveReq[保存需求到数据库]
    
    SaveReq --> Parallel[并行处理]
    
    parallel --> Classify[需求分类<br/>LLM Call #2]
    parallel --> QualityCheck[质量评估<br/>LLM Call #3]
    parallel --> UseCaseGen[用例生成<br/>LLM Call #4]
    
    Classify --> UpdateReq[更新需求分类]
    QualityCheck --> SaveQuality[保存质量评估]
    UseCaseGen --> SaveUseCase[保存用例]
    
    UpdateReq --> Merge[合并结果]
    SaveQuality --> Merge
    SaveUseCase --> Merge
    
    Merge --> ReturnResult[返回分析结果]
    ReturnResult --> End([结束])
    
    style Start fill:#e1f5fe
    style End fill:#e1f5fe
    style AIExtract fill:#fff3e0
    style Classify fill:#fff3e0
    style QualityCheck fill:#fff3e0
    style UseCaseGen fill:#fff3e0
    style Parallel fill:#f3e5f5
```

## 4. 需求状态流转图

```mermaid
stateDiagram-v2
    [*] --> DRAFT: AI提取/手动创建
    
    DRAFT --> CONFIRMED: 需求工程师确认
    DRAFT --> REJECTED: 需求工程师拒绝
    DRAFT --> DRAFT: 编辑修改
    
    CONFIRMED --> ARCHITECTURE_MAPPED: 架构映射完成
    CONFIRMED --> UNDER_REVIEW: 提交审核
    
    UNDER_REVIEW --> CONFIRMED: 审核通过
    UNDER_REVIEW --> DRAFT: 审核驳回
    
    ARCHITECTURE_MAPPED --> IMPLEMENTED: 开发实现
    IMPLEMENTED --> VERIFIED: 验证通过
    VERIFIED --> [*]
    
    REJECTED --> [*]: 删除
```

## 6. AI 追溯映射工作流程

## 8. RBAC 权限检查流程

```mermaid
flowchart TD
    Start([API 请求]) --> ExtractToken[提取 JWT Token]
    ExtractToken --> DecodeToken[解码 Token]
    DecodeToken --> VerifySignature{验证签名}
    
    VerifySignature -->|无效| Error401[401 未授权]
    VerifySignature -->|有效| GetUserId[获取用户 ID]
    
    GetUserId --> QueryDB[查询数据库]
    QueryDB --> UserExists{用户存在?}
    UserExists -->|否| Error404[404 用户不存在]
    UserExists -->|是| CheckActive{账户激活?}
    
    CheckActive -->|否| Error400[400 账户禁用]
    CheckActive -->|是| GetRole[获取用户角色]
    
    GetRole --> CheckPermission{检查权限}
    CheckPermission -->|RE| RE_Permissions[RE 权限]
    CheckPermission -->|SA| SA_Permissions[SA 权限]
    CheckPermission -->|ADMIN| ADMIN_Permissions[ADMIN 权限]
    
    RE_Permissions --> AllowRE{允许访问?}
    SA_Permissions --> AllowSA{允许访问?}
    ADMIN_Permissions --> AllowADMIN{允许访问?}
    
    AllowRE -->|是| Proceed[继续处理]
    AllowRE -->|否| Error403[403 权限不足]
    
    AllowSA -->|是| Proceed
    AllowSA -->|否| Error403
    
    AllowADMIN -->|是| Proceed
    AllowADMIN -->|否| Error403
    
    Proceed --> End([结束])
    Error401 --> End
    Error404 --> End
    Error400 --> End
    Error403 --> End
    
    style Start fill:#e1f5fe
    style End fill:#e1f5fe
    style Proceed fill:#e8f5e9
    style Error401 fill:#ffebee
    style Error403 fill:#ffebee
    style Error404 fill:#ffebee
    style Error400 fill:#ffebee
```

## 9. 数据库操作流程

```mermaid
flowchart TD
    Start([API 请求]) --> GetDBSession[获取数据库会话]
    GetDBSession --> BeginTransaction[开启事务]
    
    BeginTransaction --> ExecuteQuery[执行查询]
    ExecuteQuery --> QueryType{查询类型}
    
    QueryType -->|SELECT| SelectOp[查询数据]
    QueryType -->|INSERT| InsertOp[插入数据]
    QueryType -->|UPDATE| UpdateOp[更新数据]
    QueryType -->|DELETE| DeleteOp[删除数据]
    
    SelectOp --> CommitCheck{需要提交?}
    InsertOp --> CommitCheck
    UpdateOp --> CommitCheck
    DeleteOp --> CommitCheck
    
    CommitCheck -->|是| Commit[提交事务]
    CommitCheck -->|否| Rollback[回滚事务]
    
    Commit --> CloseSession[关闭会话]
    Rollback --> CloseSession
    
    CloseSession --> ReturnResult[返回结果]
    ReturnResult --> End([结束])
    
    style Start fill:#e1f5fe
    style End fill:#e1f5fe
    style Commit fill:#e8f5e9
    style Rollback fill:#ffebee
```

## 10. 错误处理流程

```mermaid
flowchart TD
    Start([发生异常]) --> CatchException[捕获异常]
    CatchException --> ExceptionType{异常类型}
    
    ExceptionType -->|ValidationError| Handle422[422 验证错误]
    ExceptionType -->|AuthenticationError| Handle401[401 未授权]
    ExceptionType -->|PermissionDenied| Handle403[403 禁止访问]
    ExceptionType -->|NotFound| Handle404[404 未找到]
    ExceptionType -->|LLMError| HandleLLM[LLM 服务错误]
    ExceptionType -->|DatabaseError| HandleDB[数据库错误]
    ExceptionType -->|Other| Handle500[500 服务器错误]
    
    Handle422 --> LogError[记录错误日志]
    Handle401 --> LogError
    Handle403 --> LogError
    Handle404 --> LogError
    HandleLLM --> LogError
    HandleDB --> LogError
    Handle500 --> LogError
    
    LogError --> ReturnErrorResponse[返回错误响应]
    ReturnErrorResponse --> End([结束])
    
    style Start fill:#ffebee
    style End fill:#e1f5fe
    style HandleLLM fill:#fff3e0
    style HandleDB fill:#ffebee
```
